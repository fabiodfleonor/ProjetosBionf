# Programação Aplicada à Bioinformática
Neste repositório vou reunir **trabalhos e exercícios** desenvolvidos ao longo da unidade curricular de **Programação Aplicada à Bioinformática**.
Os ficheiros incluem scripts e pequenos projetos em **Python**, focados em problemas e aplicações da **bioinformática**.
Este espaço serve apenas para guardar e mostrar a minha evolução na programação e no tratamento de dados biológicos.
